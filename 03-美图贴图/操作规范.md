# 美图贴图批量处理 — 操作规范 v3.4

## 一、核心参数（禁止擅自修改）
- TARGET_X = 2105（4096素材X中轴线对齐）
- TARGET_Y = 620（4096素材Y目标）
- **3412x5120素材：TARGET_X=2055, TARGET_Y=680，最高像素点对齐Y=630**
- DEFAULT_LEFT = 1921  DEFAULT_TOP = 590（贴图默认左上角）
- 窗口位置: (1280,0)  分辨率: 1280x1440
- Alpha阈值: >= 20（原始 Alpha 值，低于20视为透明噪点）
- **旋转手柄: 4096素材(2102,994)，3412素材(2102,970) 右拖5px**
- 必须先旋转再移动
- **y_min 噪声过滤：第一个有50个以上非透明像素的行**
- **每张图前重新查找 Meitu 窗口**

## 二、贴图位置计算（工业版 Alpha 加权中轴线）
有效像素 = Alpha >= 20
cx = sum(x * alpha) / sum(alpha)（只统计有效像素，透明度加权重心）
y_min = 有效像素最高点
dy = TARGET_Y - y_min*scale - DEFAULT_TOP
dx = TARGET_X - cx*scale - DEFAULT_LEFT
scale = 360 / 原始图片高度
