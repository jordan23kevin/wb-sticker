## 核心流程
1. 扫描 D:\Semems\0AI 下未处理的 DS* 文件夹，读取每个文件夹里的 PNG 贴图。
2. 首次启动美图秀秀并打开胚衣 D:\Semems\1胚衣\白正2.jpg。
3. 将美图窗口放到右半屏：(1280,0)，大小 1280x屏幕高度。
4. 进入 AI工具 → AI贴图 → 新增图片。
5. 通过系统文件对话框导入 PNG。
6. 必须先旋转贴图，再根据图像 alpha 有效像素计算偏移并移动到目标位置。
7. 设置混合模式/确认后 Ctrl+S 保存到当前 DS 文件夹，质量 100。
8. 保存成功后写入 D:\Semems\0AI\.processed.json，避免重复处理。

## 关键参数
- TARGET_X = 2105：贴图 X 中轴线对齐。
- TARGET_Y = 620：有效最高像素点不高于 620。
- DEFAULT_LEFT = 1921，DEFAULT_TOP = 590：贴图默认左上角。
- Alpha 阈值：原始 Alpha >= 20。
- 中轴线算法：只统计 Alpha>=20 的有效像素，并用 Alpha 加权重心 cx = sum(x*alpha)/sum(alpha)，不再用左右边界框中心。
- scale = 360 / 原始图片高度。
- dx = TARGET_X - cx * scale - DEFAULT_LEFT。
- dy = TARGET_Y - y_min * scale - DEFAULT_TOP。
- 旋转手柄：相对坐标 (822,975)，拖到 (827,975)，即绝对 (2102,975) → (2107,975)。
