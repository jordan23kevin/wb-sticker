# WB 贴图 — 快速运行手册

## 触发词

用户说 **"WB 贴图"**，立即运行

## 运行命令

```bash
# 白T（自动扫描全部 DX 文件夹）
uv run --with pillow --with numpy --with pyperclip --with pywin32 python wb_sticker.py

# 黑T
uv run --with pillow --with numpy --with pyperclip --with pywin32 python wb_sticker.py --black

# 指定文件夹
uv run --with pillow --with numpy --with pyperclip --with pywin32 python wb_sticker.py DX0020
```

## 停止

用户说 **"停止"**，立即杀进程：

```bash
powershell -NoProfile -ExecutionPolicy Bypass -Command '$targets = Get-CimInstance Win32_Process | Where-Object { $_.CommandLine -and ($_.Name -in @("python.exe","python3.exe","py.exe","uv.exe","bash.exe")) -and ( $_.CommandLine -like "*wb_sticker*" -or $_.CommandLine -like "*WB贴图*" -or $_.CommandLine -like "*贴图*" -or $_.CommandLine -like "*XiuXiu*" ) }; if ($targets) { $targets | ForEach-Object { try { Stop-Process -Id $_.ProcessId -Force -ErrorAction Stop; Write-Output ("KILLED " + $_.ProcessId + " " + $_.Name) } catch { Write-Output ("FAILED " + $_.ProcessId + " " + $_.Exception.Message) } } } else { Write-Output "NO_MATCHING_SCRIPT_PROCESS" }'
```

## 前置条件

1. 美图秀秀已安装
2. DX 文件夹有 PNG 素材
3. 胚衣在 `D:\Semems\1胚衣\`
