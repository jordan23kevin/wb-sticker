# WB 贴图 — 快速运行手册 v2.4.0

## 触发词

用户说 **"WB 贴图"**

## 运行命令

```bash
# 白T 指定文件夹
uv run --with pillow --with numpy --with pyperclip --with pywin32 python wb_sticker.py DX0020

# 黑T 指定文件夹
uv run --with pillow --with numpy --with pyperclip --with pywin32 python wb_sticker.py --black DX0020
```

## 停止

用户说 **"x"** 或 **"停止"**，立即杀进程。

## 前置条件

1. 美图秀秀已安装
2. DX 文件夹有 PNG 素材（由 WB去背生成）
3. 胚衣在 `D:\Semems\1胚衣\`
