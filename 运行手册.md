# WB 贴图 — 快速运行手册 v2.7.0

## 触发词
用户说 **"WB 贴图"**

## 执行顺序（重要）
1. **先跑白T**（默认参数）
2. 白T完成后**立即跑黑T**（`--black`）
3. **全部做完再汇报**，不要中途汇报

## 运行命令
```bash
# 第1步：白T 自动扫描全部DX文件夹
python wb_sticker.py

# 第2步：白T完成后，黑T 自动扫描全部DX文件夹
python wb_sticker.py --black
```

## 关键注意事项
- ⚠️ **胚衣重开**：每次首次打开美图秀秀置入胚衣后，必须关闭胚衣再置入一次，否则图片大小不对
- ⚠️ **顺序**：先白T后黑T，不要颠倒
- ⚠️ **汇报**：两个都完成后统一汇报

## 停止
用户说 **"x"** 或 **"停止"**，立即杀进程。
